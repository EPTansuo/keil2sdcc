convert keil code to sdcc

support
- c51 to sdcc


## install
```
git clone 
python setup.py install
```

## usage
import use
```python
import keil2sdcc
keil2sdcc.C512SDCC(keil_file)
```

console use
```sh
keil2sdcc keil_file1 keil_file2
```
